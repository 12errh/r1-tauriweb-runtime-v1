## Phase 1 — WASI Shim: SQLite Syscall Completion

> **Goal**: The WASI shim implements every syscall SQLite needs.
> After this phase, `rusqlite` can open a database file stored in OPFS.

### What SQLite Needs That May Be Missing

Based on the Phase 0 audit, these are the most likely gaps:

**`fd_seek` — incomplete implementation**
SQLite uses all three seek modes. The current shim may only implement `SEEK_SET`:
```typescript
// SEEK_SET = 0  (seek from beginning)
// SEEK_CUR = 1  (seek from current position)  ← likely missing
// SEEK_END = 2  (seek from end)                ← likely missing
```

**`fd_filestat_get` — file metadata**
SQLite checks file sizes before reads. Returns inode, size, type, timestamps.

**`fd_fdstat_get` — file descriptor status**
SQLite checks whether an fd is readable/writable before operations.

**`fd_sync` — flush to storage**
SQLite calls this after writes. Currently may be a no-op stub.

**`path_filestat_get` — stat by path**
SQLite checks if a WAL (Write-Ahead Log) or journal file exists.

**`path_rename` — atomic rename**
SQLite uses atomic rename for its journaling system.

### Agent Instructions

**Step 1.1 — Implement complete `fd_seek`**
```typescript
'fd_seek': (fd: number, offset_low: number, offset_high: number,
            whence: number, newoffset_ptr: number): number => {
  const file = this.fdTable.get(fd);
  if (!file) return ERRNO.BADF;

  const offset = BigInt(offset_low) | (BigInt(offset_high) << 32n);

  switch (whence) {
    case 0: // SEEK_SET
      file.position = Number(offset);
      break;
    case 1: // SEEK_CUR
      file.position = file.position + Number(offset);
      break;
    case 2: // SEEK_END
      file.position = file.data.length + Number(offset);
      break;
    default:
      return ERRNO.INVAL;
  }

  file.position = Math.max(0, file.position);

  // Write new position back to WASM memory
  const view = new DataView(this.memory.buffer);
  view.setBigInt64(newoffset_ptr, BigInt(file.position), true);
  return ERRNO.SUCCESS;
}
```

**Step 1.2 — Implement `fd_filestat_get`**
Returns file metadata. SQLite uses `st_size` to know file length:
```typescript
'fd_filestat_get': (fd: number, buf_ptr: number): number => {
  const file = this.fdTable.get(fd);
  if (!file) return ERRNO.BADF;

  const view = new DataView(this.memory.buffer);
  // struct filestat layout (64 bytes):
  // 0:  dev (u64)
  // 8:  ino (u64)
  // 16: filetype (u8)
  // 24: nlink (u64)
  // 32: size (u64)      ← SQLite reads this
  // 40: atim (u64)
  // 48: mtim (u64)
  // 56: ctim (u64)

  view.setBigUint64(buf_ptr + 0,  0n, true);           // dev
  view.setBigUint64(buf_ptr + 8,  BigInt(fd), true);   // ino
  view.setUint8(buf_ptr + 16, 4);                       // filetype: REGULAR_FILE
  view.setBigUint64(buf_ptr + 24, 1n, true);            // nlink
  view.setBigUint64(buf_ptr + 32, BigInt(file.data.length), true); // size
  const now = BigInt(Date.now()) * 1_000_000n;
  view.setBigUint64(buf_ptr + 40, now, true);           // atim
  view.setBigUint64(buf_ptr + 48, now, true);           // mtim
  view.setBigUint64(buf_ptr + 56, now, true);           // ctim

  return ERRNO.SUCCESS;
}
```

**Step 1.3 — Implement `fd_fdstat_get`**
```typescript
'fd_fdstat_get': (fd: number, buf_ptr: number): number => {
  const file = this.fdTable.get(fd);
  if (!file) return ERRNO.BADF;

  const view = new DataView(this.memory.buffer);
  view.setUint8(buf_ptr, 4);       // fs_filetype: REGULAR_FILE
  view.setUint16(buf_ptr + 2, 0, true); // fs_flags
  view.setBigUint64(buf_ptr + 8,
    0x40n | 0x80n | 0x200n | 0x400n, true); // fs_rights_base (READ|WRITE|SEEK|TELL)
  view.setBigUint64(buf_ptr + 16, 0n, true); // fs_rights_inheriting

  return ERRNO.SUCCESS;
}
```

**Step 1.4 — Implement `fd_sync`**
SQLite calls this after writes. Flush the fd's data to OPFS:
```typescript
'fd_sync': (fd: number): number => {
  const file = this.fdTable.get(fd);
  if (!file || !file.path) return ERRNO.BADF;

  // Write current buffer to VFS (async, fire and forget)
  this.vfs.write(file.path, file.data).catch(console.error);

  return ERRNO.SUCCESS;
}
```

**Step 1.5 — Implement `path_filestat_get`**
```typescript
'path_filestat_get': (
  dirfd: number, flags: number,
  path_ptr: number, path_len: number,
  buf_ptr: number
): number => {
  const path = this.readString(path_ptr, path_len);
  const resolvedPath = this.resolvePath(dirfd, path);

  // Check cache first for fast response
  const cached = this.vfs.getCached(resolvedPath);
  if (cached === null) return ERRNO.NOENT;

  const view = new DataView(this.memory.buffer);
  view.setBigUint64(buf_ptr + 0,  0n, true);
  view.setBigUint64(buf_ptr + 8,  0n, true);
  view.setUint8(buf_ptr + 16, cached ? 4 : 3); // 4=file, 3=dir
  view.setBigUint64(buf_ptr + 24, 1n, true);
  view.setBigUint64(buf_ptr + 32, BigInt(cached?.length ?? 0), true);
  const now = BigInt(Date.now()) * 1_000_000n;
  view.setBigUint64(buf_ptr + 40, now, true);
  view.setBigUint64(buf_ptr + 48, now, true);
  view.setBigUint64(buf_ptr + 56, now, true);

  return ERRNO.SUCCESS;
}
```

**Step 1.6 — Implement `path_rename`**
SQLite uses this for its atomic write pattern:
```typescript
'path_rename': (
  old_fd: number, old_path_ptr: number, old_path_len: number,
  new_fd: number, new_path_ptr: number, new_path_len: number
): number => {
  const oldPath = this.resolvePath(old_fd, this.readString(old_path_ptr, old_path_len));
  const newPath = this.resolvePath(new_fd, this.readString(new_path_ptr, new_path_len));

  // Read old, write new, delete old — atomic from SQLite's perspective
  const data = this.vfs.getCached(oldPath);
  if (data === null) return ERRNO.NOENT;

  this.vfs.write(newPath, data).catch(console.error);
  this.vfs.delete(oldPath).catch(console.error);

  return ERRNO.SUCCESS;
}
```

~~Phase 2 — SQLite via WASM~~
REPLACED BY: SQLite Fix Roadmap (see Sqlite fix roadmap.md)
Uses @sqlite.org/sqlite-wasm — no C compilation required.

> **Goal**: `rusqlite` with the `bundled` feature compiles to WASM and opens
> a database file stored in OPFS. The developer's Rust SQL code is unchanged.

### How It Works

```
Developer's Rust:
  let conn = Connection::open("/app/data/app.db")?;
  conn.execute("INSERT INTO notes VALUES (?1)", [&text])?;
         ↓
WASM compilation (wasm-pack with rusqlite --features bundled)
         ↓
WASI shim intercepts path_open("/app/data/app.db")
         ↓
VFS creates/reads /app/data/app.db in OPFS
         ↓
SQLite reads and writes the file as bytes
         ↓
Data persists across page refreshes
```

`src/lib.rs`:
```rust
use wasm_bindgen::prelude::*;
use rusqlite::{Connection, Result};

fn get_conn() -> Result<Connection> {
    Connection::open("/app/data/test.db")
}

#[wasm_bindgen]
pub fn create_table(payload: &str) -> String {
    match get_conn() {
        Ok(conn) => {
            match conn.execute(
                "CREATE TABLE IF NOT EXISTS items (id INTEGER PRIMARY KEY, value TEXT)",
                []
            ) {
                Ok(_) => serde_json::json!({ "ok": true }).to_string(),
                Err(e) => serde_json::json!({ "error": e.to_string() }).to_string(),
            }
        },
        Err(e) => serde_json::json!({ "error": e.to_string() }).to_string(),
    }
}

#[wasm_bindgen]
pub fn insert_item(payload: &str) -> String {
    #[derive(serde::Deserialize)]
    struct Args { value: String }

    let args: Args = match serde_json::from_str(payload) {
        Ok(a) => a,
        Err(e) => return serde_json::json!({ "error": e.to_string() }).to_string(),
    };

    match get_conn() {
        Ok(conn) => {
            match conn.execute("INSERT INTO items (value) VALUES (?1)", [&args.value]) {
                Ok(id) => serde_json::json!({ "ok": conn.last_insert_rowid() }).to_string(),
                Err(e) => serde_json::json!({ "error": e.to_string() }).to_string(),
            }
        },
        Err(e) => serde_json::json!({ "error": e.to_string() }).to_string(),
    }
}

#[wasm_bindgen]
pub fn get_items(payload: &str) -> String {
    match get_conn() {
        Ok(conn) => {
            let mut stmt = match conn.prepare("SELECT id, value FROM items") {
                Ok(s) => s,
                Err(e) => return serde_json::json!({ "error": e.to_string() }).to_string(),
            };

            let items: Vec<serde_json::Value> = stmt
                .query_map([], |row| {
                    Ok(serde_json::json!({
                        "id": row.get::<_, i64>(0)?,
                        "value": row.get::<_, String>(1)?
                    }))
                })
                .unwrap()
                .filter_map(|r| r.ok())
                .collect();

            serde_json::json!({ "ok": items }).to_string()
        },
        Err(e) => serde_json::json!({ "error": e.to_string() }).to_string(),
    }
}
```

```typescript
import { describe, it, expect, beforeEach } from 'vitest';
import { WasmOrchestrator } from './wasm-orchestrator';
import { VFS } from './vfs';

describe('Phase 2: SQLite via WASM', () => {
  let orchestrator: WasmOrchestrator;
  let vfs: VFS;

  beforeEach(async () => {
    vfs = new VFS();
    await vfs.init();
    orchestrator = new WasmOrchestrator(vfs);

  });

    expect(result).toHaveProperty('ok');
    expect(typeof result.ok).toBe('number');
```

**Step 4.4 — Implement `patch-build-rs.ts`**
```typescript
export async function patchBuildRs(root: string): Promise<void> {
  const buildRsPath = path.join(root, 'src-tauri', 'build.rs');

  if (!fs.existsSync(buildRsPath)) return; // no build.rs, nothing to do

  const current = fs.readFileSync(buildRsPath, 'utf-8');

  // Already patched
  if (current.trim() === 'fn main() {}') return;

  // Back up original
  fs.writeFileSync(buildRsPath + '.r1-backup', current);

  // Replace with empty main
  fs.writeFileSync(buildRsPath, 'fn main() {}\n');
}
```

**Step 4.5 — Implement `patch-cargo.ts`**
This is the most complex patch. It must:
1. Parse the existing `Cargo.toml`
2. Remove `[build-dependencies]` section
3. Add `[lib]` section with correct `name` and `crate-type`
4. Add `wasm-bindgen`, `serde`, `serde_json` to `[dependencies]`
5. Move `tauri` and all other native-only deps to
   `[target.'cfg(not(target_arch = "wasm32"))'.dependencies]`

Use `@iarna/toml` for parsing:
```typescript
import TOML from '@iarna/toml';

export async function patchCargo(root: string): Promise<void> {
  const cargoPath = path.join(root, 'src-tauri', 'Cargo.toml');
  const raw = fs.readFileSync(cargoPath, 'utf-8');
  const cargo = TOML.parse(raw);

  // Get crate name from package.name, convert to snake_case
  const crateName = (cargo.package as any).name.replace(/-/g, '_');

  // Add [lib] section
  (cargo as any).lib = {
    name: crateName,
    'crate-type': ['cdylib', 'rlib']
  };

  // Add WASM deps
  (cargo.dependencies as any)['wasm-bindgen'] = '0.2';
  (cargo.dependencies as any)['serde'] = { version: '1', features: ['derive'] };
  (cargo.dependencies as any)['serde_json'] = '1';

  // Move native deps
  const nativeDeps: Record<string, unknown> = {};
  const nativeOnly = ['tauri', 'tauri-plugin-opener', 'tauri-plugin-store'];

  for (const dep of nativeOnly) {
    if ((cargo.dependencies as any)[dep]) {
      nativeDeps[dep] = (cargo.dependencies as any)[dep];
      delete (cargo.dependencies as any)[dep];
    }
  }

  // Write target-specific deps
  if (Object.keys(nativeDeps).length > 0) {
    (cargo as any)["target.'cfg(not(target_arch = \"wasm32\"))'.dependencies"] = nativeDeps;
  }

  // Remove [build-dependencies]
  delete (cargo as any)['build-dependencies'];

  // Write back
  fs.writeFileSync(cargoPath + '.r1-backup', raw);
  fs.writeFileSync(cargoPath, TOML.stringify(cargo as any));
}
```

**Step 4.6 — Implement `rewrite-rust.ts`**
This rewrites Rust commands from the Tauri pattern to the R1 JSON contract.
This is the most valuable part of the CLI:

```typescript
// BEFORE (what the developer wrote):
#[tauri::command]
fn greet(name: String) -> String {
    format!("Hello, {}!", name)
}

// AFTER (what R1 needs):
#[wasm_bindgen]
pub fn greet(payload: &str) -> String {
    #[derive(serde::Deserialize)]
    struct Args { name: String }
    let args: Args = match serde_json::from_str(payload) {
        Ok(a) => a,
        Err(e) => return serde_json::json!({ "error": e.to_string() }).to_string(),

}
```

The rewriter must:
1. Find all functions annotated with `#[tauri::command]`
2. Extract the function name, parameter names and types, return type
3. Generate a `#[derive(Deserialize)]` struct for the input
4. Rewrite the function signature to `(payload: &str) -> String`
5. Insert JSON decode at the top
6. Wrap the return value in `serde_json::to_string`
7. Add required `use` statements at the top of the file
8. Add the `#[cfg(not(target_arch = "wasm32"))]` gate on the `run()` function

**Step 4.7 — Implement `patch-vite.ts`**
```typescript
export async function patchVite(root: string): Promise<void> {
  const vitePath = path.join(root, 'vite.config.ts');
  if (!fs.existsSync(vitePath)) return;

  let content = fs.readFileSync(vitePath, 'utf-8');

  // Already patched
  if (content.includes('@r1/vite-plugin')) return;

  // Add import
  content = `import { r1Plugin } from '@r1/vite-plugin';\n` + content;

  // Add to plugins array — find plugins: [ and insert r1Plugin()
  content = content.replace(
    /plugins:\s*\[/,
    `plugins: [\n    r1Plugin({ rustSrc: './src-tauri' }),`
  );

  fs.writeFileSync(vitePath, content);
}
```

```rust
use proc_macro::TokenStream;
use quote::quote;
use syn::{parse_macro_input, ItemFn, FnArg, ReturnType, Type};

#[proc_macro_attribute]
pub fn command(_attr: TokenStream, item: TokenStream) -> TokenStream {
    let input = parse_macro_input!(item as ItemFn);
    let fn_name = &input.sig.ident;
    let fn_body = &input.block;

    // Extract parameter names and types
    let params: Vec<_> = input.sig.inputs.iter().filter_map(|arg| {
        if let FnArg::Typed(pat_type) = arg {
            Some(pat_type)
        } else {
            None
        }
    }).collect();

    // Generate the Args struct fields
    let struct_fields = params.iter().map(|p| {
        let name = &p.pat;
        let ty = &p.ty;
        quote! { #name: #ty }
    });

    // Generate the destructuring
    let destructure = params.iter().map(|p| {
        let name = &p.pat;
        quote! { let #name = args.#name; }
    });

    // Generate the return wrapping
    let expanded = quote! {
        #[wasm_bindgen::prelude::wasm_bindgen]
        pub fn #fn_name(payload: &str) -> String {
            #[derive(serde::Deserialize)]
            #[allow(non_snake_case)]
            struct __R1Args {
                #(#struct_fields),*
            }

            let args: __R1Args = match serde_json::from_str(payload) {
                Ok(a) => a,
                Err(e) => return serde_json::json!({ "error": e.to_string() }).to_string(),
            };

            #(#destructure)*

            let __r1_result = (|| -> _ { #fn_body })();

            match serde_json::to_string(&__r1_result) {
                Ok(s) => s,
                Err(e) => serde_json::json!({ "error": e.to_string() }).to_string(),
            }
        }
    };

    TokenStream::from(expanded)
}
```
